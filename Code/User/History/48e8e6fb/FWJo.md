# NAT and port forwarding

## NAT

- Responsible for converting IP addresses as they pass over the boundary between public and private spaces

## NAT steps

- The router in the organization will place an entry into its network translation table

- Records where the IP address and the port number of computer sending the packet

- The router records its own IP address

- Records a random port number

- Uses the random port number generated to direct packets to the computer on the internal network
